## Installation

git clone [https://github.com/kechinval/simple-chat.git](https://github.com/kechinval/simple-chat.git)<br>
cd simple-chat<br>
npm install<br>
cd server<br>
npm install<br>

## To start project

In the ./ directory, you should run:

### `npm start`

In the ./server directory, you should run:

### `nodemon app`
